## üîë **API Keys Required**

### **Essential APIs**
1. **Google Gemini API Key**
   - Purpose: Text embeddings (768 dimensions)
   - Get from: https://aistudio.google.com/app/apikey
   - Cost: Free tier available, pay-per-use
   - Usage: ~$0.01-0.05 per document

2. **Groq API Key**
   - Purpose: Chat completions (LLM responses)
   - Get from: https://console.groq.com/keys
   - Cost: Free tier available, very low cost
   - Usage: ~$0.001-0.01 per query

### **Optional APIs**
- **OpenAI API Key**: Alternative to Groq (higher cost)
- **Anthropic Claude API**: Alternative LLM provider
